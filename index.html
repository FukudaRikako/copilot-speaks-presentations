TTS PRESENTATION

プレゼンを声で届ける。

Text-to-Speech でスライドを自動読み上げ。
声を選び、感情を込め、時間に合わせる — すべてブラウザだけで。

WHY TTS?

プレゼンの悩み、ありませんか？

😰
人前で話すのが苦手

⏱️
時間配分が難しい
5分と言われたのに3分で終わった…
🌍
多言語での発表
英語プレゼンの発音に自信がない
♿
アクセシビリティ

TECHNOLOGY

🔧 技術のしくみ

ブラウザ内蔵の Web Speech API で動作。
さらに高品質な音声が必要なら Azure へ。

🌐
Web Speech API
ブラウザに内蔵された無料の音声エンジン。
インストール不要、APIキー不要。
このアプリはこれだけで動いています。
☁️
Azure Text to Speech
マイクロソフトのクラウド音声サービス。
400以上のニューラル音声、感情表現、SSML対応。
プロ品質の音声が必要な場合に。
FEATURES

3つの魔法

ブラウザの Web Speech API だけで、ここまでできる。

🎙️
声を選ぶ
日本語・英語・多言語の音声から好みの声を選択
🎭
感情を込める
落ち着き・情熱・やさしさなど6種類のプリセット
⏰
時間に合わせる
目標時間を設定すれば最適な読み上げ速度を自動計算
FEATURE 01

🎙️ 声を選ぶ

右上の⚙ボタンから音声設定を開いて、好みの声を試してみてください。

この文章が、選んだ声で読み上げられます。日本語でも英語でも、ブラウザが対応するすべての言語の音声が使えます。お気に入りの声を見つけてみてください。

FEATURE 02

🎭 感情を込める

ピッチと音量、スピードの組み合わせで、声に表情をつけます。

FEATURE 03

⏰ 時間に合わせる

「5分のプレゼンにしたい」と設定すれば、全スライドの文字数から最適な読み上げ速度を自動計算。練習いらずで時間ピッタリ。

全スライドの総文字数から自動で速度を調整します
